Variables agregadas a Objetos

ExclusivoClase

Al igual que en los hechizos, esta variable sirve para que solo funcione el objeto si la utiliza una clase especifica, es especial para usar en las armas.
ExclusivoClase=CLERIGO

NoSeCae

Esta variable puede tomar solo dos valores 0 y 1, y determina si el objeto se cae o no, si es 1 no se cae y si es 0 o no existe se cae.

Ejemplo: NoSeCae=1

NoSeVende

Esta variable solo toma dos valores 0 y 1, y sirve para que los NPC que puedan comprarla no la pongan a la venta. Si es un item valioso y difícil de conseguir es recomendable utilizar esta variable, porque por ejemplo, en vendedor de Banderbill es capaz de vender después nuestro item, por eso existe esta variable.

NoSeVende=1

SoloNPC

Si a un arma, le colocamos esta variable, que puede ser 0 o 1, si es 1 el arma solo podrá atacar a un NPC, no servirá para atacar usuarios.
SoloNPC=1

Paraliza

Si es un arma, podemos determinan las probabilidades de paralizar del arma. Valores del 0 al 100.
Paraliza=40

NoParalisis

Si es una armadura, podemos indicarles las probabilidades de resistirse a ser paralizado que tiene la armadura. Valores del 0 al 100.
NoParalisis=20

Devuelve

Solo para escudos o armaduras. Se trata de un valor del 0 al 100, que indica si el escudo devuelve un ataque mágico o no, el valor q contiene es el porcentaje q devolverá al atacante, a su vez también protege al atacado en la mitad de lo q devolvió. Ejemplo:
Devuelve=1.4
Eso significa q devolverá el 1.4% del daño, y protegerá al atacado por 0.7% del daño.
Minimo Valor: 0 / Maximo Valor: 100

NumCabeza

Utilizable solo en ropas, nos permitirá cambiar la cabeza a la vez q el body. Para hacer una ropa q nos quite la cabeza, no podremos usar NumCabeza=0, tendremos q utilizar NumCabeza=-1 para poder hacer q no tenga cabeza, después para ponerle otra cabeza, temporalmente, o sea, es solo por usar esta ropa, no es permanente. NumCabeza=11

Objetos Especiales

...
[OBJx]
...
Poder=1.2
Magic=1.4
Agilidad=1.2
...

Ahora disponemos de estas tres variables nuevas, que nos permitirar crear objetos que nos suban o bajen, poder, magia y agilidad.

Poder - Nos modifica el daño con el que pegamos con arco u espada.
Magic - Nos modifica el daño de las magias.
Agilidad - Nos da mas evacion a la hora de recibir un golpe

Minimo de Nivel para Poder Usarse/Equiparse 

...
[OBJx]
...
MinNivel=15
...

En este ejemplo, el objeto x solo podra ser utilizado despues de tener el minimo de Nivel o superior.
En este ejemplo, Nivel 15.

Baculos

OBJ.dat

[OBJx]
...
Objtype=2
...
StaffPower=2
StaffDamageBonus=30
...

Objtype=2, indica que es un arma.
StaffPower=2, indica que el nivel del baculo es 2.
StaffDamageBonus=30, indica que el bonus de daño es el 36%.

Defensa Magica 

Solo utilizable en Cascos, Herramientas, Armaduras y Accesorios.
El item equipado, con estos campos, nos reducira el daño de cualquier ataque magico proveniente de un Usuario o de un NPCs.

OBJ.dat

[OBJx]
...
DefensaMagicaMin=20
DefensaMagicaMax=25
...

Indica que nos protejera de magia, de 20 a 25 puntos de daño.

El valor minimo es 0, y el maximo 32000.
